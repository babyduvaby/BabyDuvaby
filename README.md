# Baby Duvaby Landing

Landing page mobile-first en React + Tailwind CSS para Baby Duvaby.

## Requisitos
- Node.js 18+
- npm 9+

## Ejecutar en local
```bash
npm install
npm start
```

## Build de producción
```bash
npm run build
```

## Panel administrador
- URL: `https://baby-duvaby.vercel.app/admin/login`
- El panel ya no se muestra dentro de la landing pública.
