# Baby Duvaby Landing

Landing page mobile-first en React + Tailwind CSS para Baby Duvaby.

## Requisitos
- Node.js 18+
- npm 9+

## Ejecutar en local
```bash
npm install
npm start
```

## Build de producción
```bash
npm run build
```

## Panel administrador
- Contraseña inicial: `duvaby2026`
- Permite editar textos, imágenes y WhatsApp.
- Los cambios y el contador de clics se guardan en `localStorage`.
